# MAX Studio — студия звукозаписи

Лендинг с акцентом на визуальный опыт и внимание к деталям.

🔗 [Демо](https://smehovartem566.github.io/max-studio/)

## 🎨 Дизайнерские решения

- Полностью кастомный дизайн (не шаблон)
- Плавные анимации появления элементов при скролле
- Декоративные волны и анимированные круги на фоне
- Карточки услуг с визуальным выделением популярного тарифа
- Аудиоплеер с собственным оформлением
- Адаптив под все устройства

## 🛠 Технологии

- HTML5, CSS3
- Bootstrap 5 (сетка, компоненты)
- Немного JavaScript
- Figma (макет)

## 📱 Адаптивность

- Desktop → планшеты → телефоны
- Бургер-меню на мобильных
- Кастомные медиазапросы

## ✨ Особенности

- Анимированные волны в hero-секции
- Появление блоков при скролле (animation-timeline: view)
- Собственный плеер с плейлистом
- Форма с валидацией и модальным окном
